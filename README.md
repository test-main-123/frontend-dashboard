# Frontend Dashboard

A sample frontend dashboard application built with React.

## Setup

```bash
npm install
```

## Development

```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the dashboard.

## Project Structure

- `src/index.js` - Application entry point
- `src/App.js` - Main dashboard component
- `src/utils.js` - Utility functions for formatting and calculations

## Features

- Revenue tracking with growth indicators
- Active user metrics
- Order volume display
